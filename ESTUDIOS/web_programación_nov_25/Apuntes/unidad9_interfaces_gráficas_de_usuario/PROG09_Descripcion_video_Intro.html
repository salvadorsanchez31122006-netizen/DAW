Este vídeo está dedicado a introducir la tecnología JavaFX.

En esta parte se estudia el HolaMundo prediseñado para JavaFX y otro ejemplo más elaborado llamado HolaEinstein.

Se explica que cualquier aplicación JavaFX siempre herada de la clase abstracta Application.

Esa clase tiene un método que se debe sobrescribir siempre y es el metodo start(), la puerta inicial de la aplicación.

Se añade en start() un poco de código para que se escriba Hola Mundo.

Posteriormente se muestran y se ejecutan mediante NetBeans
