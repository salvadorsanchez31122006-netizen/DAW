Se visualiza cómo crear una aplicación donde se meuve una bola a través de una pequeña cuadrícula, similar a un trablero.

Dicho tablero se ahce empleando una panel de contenido, un GridPane.

Se emplean cuatro botones que corresponden a las flechas apra mover a derecha, izquierda, arriba y abajo la bola, a través del tablero.

Para los botones se usan Button de JavaFX.

Para realizar la bola se dibuja mediante las formas shapes de JavaFX
