Descripción de la animación sobre como añadir un driver JDBC para MySQL en NetBeans

Se muestra una animación en la que se van sucediendo los siguientes aspectos.

  1. En primer primer lugar se muestra que la página donde se puede descargar el driver es en la página https://www.mysql.com/products/connector/
  2. En segundo lugar se muestra una captura de pantalla donde se indica que se debe seleccionar la versión del conector para Java.
  3. Después se muestra que debe elegirse la opción de descargar la versión independiente de plataforma.
  4. A continuación se muestra que debe hacerse clic en la opción que pone "No thanks, just download", para iniciar la descarga sin necesidad de registro.

Después de los pasos anteriores se indica que se debe descomprimir el archivo descargado y localizar el archivo .jar que hay en su interior llamado mysql-connector-java-8.0.20.jar.

Los siguientes pasos explican como incorporar la librería anterior a un proyecto java con librerías:

  1. Se indica que se debe crear un proyecto de NetBeans que tenga la opción "Use dedicated folder for storing libraries".
  2. Se incorpora una nueva librería, en las sección Libraries del proyecto, seleccionado la opción "Add Jar/Folder".
  3. Por último, se selecciona el archivo jar del conector para MySQL y se marca la opción "Copy to Libraries Folder".
