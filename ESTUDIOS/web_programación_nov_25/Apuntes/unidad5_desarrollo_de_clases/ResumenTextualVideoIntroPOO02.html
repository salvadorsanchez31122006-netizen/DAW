Se continúa con la introducción a la programación orientada a objetos:

  • Introducción a la terminología de la programación orientada a objetos: clase, objeto, instancia, instanciación, modularización, encapsulación, herencia, polimorfismo.
  • Se usan los coches como ejemplo de la vida real para modelar los objetos.
  • Se define el concepto de clase como el modelo donde se redactan las características comunes de un grupo de objetos mediante el ejemplo de los coches, donde cada coche concreto es un ejemplar o insancia de la clase.
  • Ejemplos de propiedades (o atributos) y comportamientos en los objetos de tipo Coche, es decir, de la clase Coche, que establece las características comunes a todos los objetos coche.
  • Una vez que se ha desarrollado el ejemplo del coche se implementa en Java la clase Coche utilizando el entorno Netbeans donde se van declarando una a una las propiedades que ya se han definido.
  • Tras la declaración de los atributos se implementa un método constructor básico y se implementa una clase de prueba con un método main para poder probar la clase Coche que se acaba de implementar donde se instancia un objeto de la clase Coche.
