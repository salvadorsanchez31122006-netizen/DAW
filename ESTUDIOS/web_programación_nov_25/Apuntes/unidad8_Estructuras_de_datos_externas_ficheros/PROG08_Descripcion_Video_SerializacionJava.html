Se visualiza la creación en un entorno de programación, de una aplicación sobre serialización de objetos. Se crea una clase contacto con nombre y apellidos. Se emplean las clases ObjectInputStream y ObjectOutputStream, para leer y escribir datos respectivamente, de tipo contacto. El objeto ObjectInputStream envuelve a un objeto FileInputStream. Con esa envoltura se realiza una lectura, con el método readObject(). De manera similar se hace con FileOutoputStream, que es envuelta por ObjectOutputStream, y de este modo se utiliza el método writeObject() para escribir objetos en el fichero.
