Se visualiza cómo crear y acceder a carpetas en MS-DOS, mediante los comandos md y cd respectivamente. Dichos comandos se escriben en la línea de comandos de MS-DOS, a la que hemos accedido desde Windows, en el apartado Ejecutar… y luego escribiendo cmd. Una vez que se accede a una carpeta se puede editar un fichero escribiendo edit y el nombre del fichero. Una vez guardado, se puede ver el contenido del fichero con el comando type.

También si se crean carpetas dentro de otras, se puede ver la estructura escribiendo el comando tree.

También se comenta el comando net users, que sirve para cosas como crear nuevos usuarios en el sistema.
